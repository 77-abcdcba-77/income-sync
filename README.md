# 项目收支统计系统

本系统用于管理项目订单、客户付款、预算追加和工具费用支出，并支持从 Excel 表格导入更新数据、自动保存历史版本、回退到任意历史快照。

## 主要功能

- 录入和编辑订单：微信号、任务名字、单号、接单日期、截止时间/状态、基础价格。
- 记录客户付款流水，自动汇总已付、剩余未收、每日收入、每月收入。
- 记录预算追加流水，自动更新订单总价。
- 记录工具费用支出，自动计算净收入。
- 导入 Excel 明细表，自动匹配现有订单并更新数据。
- 每次确认导入前自动保存当前数据库快照，可在页面回退历史版本。
- 支持订单筛选、收款提醒、客户排行、状态分布和图表看板。
- **未付款订单概览**：可视化展示所有待收订单，分为"部分付款"（已付一部分）和"未付款"（完全未付）两类，点击卡片可直接跳转到该订单的付款流水详情。
- 支持导出美化后的 Excel：`/export/xlsx`。

## 运行方式

### 方式一：双击启动

在项目目录中双击：

```text
一键启动.bat
```

启动后浏览器访问：

```text
http://127.0.0.1:5050
```

### 方式二：命令行启动

先进入项目目录：

```powershell
cd c:\Users\11145\Desktop\work\统计系统\project_income_system
```

如果已经有 `.venv`，直接运行：

```powershell
.\.venv\Scripts\python.exe app.py
```

然后访问：

```text
http://127.0.0.1:5050
```

### 方式三：备用开发端口

如果 `5050` 端口被占用，可以运行备用启动脚本：

```powershell
.\.venv\Scripts\python.exe run_dev_server.py
```

然后访问：

```text
http://127.0.0.1:5051
```

## 首次安装依赖

如果 `.venv` 不存在，先创建虚拟环境并安装依赖：

```powershell
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

## 导入 Excel

1. 打开系统首页。
2. 在“Excel 导入与版本回退”区域选择 `.xlsx` 表格。
3. 点击“预览导入”，确认新增、更新、不变和付款校正数量。
4. 点击“确认导入”后，系统会先自动保存当前数据库快照，再更新数据。

当前支持的明细表列名包括：

```text
微信号、任务名字、单号、截止时间、价格、到账、剩余
```

## 未付款订单概览

页面中部的"未付款订单概览"面板将所有待收订单分为两类可视化展示：

- **部分付款**：已收到部分付款，但仍有尾款未结清的订单。橙色标题栏，显示剩余待收总额。
- **未付款**：尚未收到任何付款的订单。红色标题栏，显示订单总价。

每张订单卡片展示客户名、任务名、剩余金额（红色高亮）、已付比例进度条、总价和已付金额。顶部选项卡支持切换"全部待收 / 部分付款 / 未付款"三种视图。

**点击任意订单卡片**，页面会自动滚动到下方的付款操作区域，并自动选中该订单，同时加载其付款流水和预算记录，方便直接操作。

## 历史回退

每次确认导入前，系统都会把导入前的数据复制一份到数据库历史版本中。

在页面“历史版本”区域点击“回退”即可恢复到对应版本。回退前系统也会再自动保存一份当前状态，方便撤销回退。

## 数据文件

主要数据存放在：

```text
data/records.db
```

请不要随意删除这个文件。历史版本、导入批次、订单、付款、预算和支出都会保存在这个数据库中。

## 导出 Excel

页面右上角点击“导出 XLSX”，或直接访问：

```text
http://127.0.0.1:5050/export/xlsx
```
